<TLDR>
The Portfolio dashboard shows everything about your money: total value, active positions across all protocols, yield earned over time, transaction history, and multi-asset breakdowns. It's your single source of truth for all DeFi activity on Tasmil.
</TLDR>

## Dashboard Overview

<StepScreenshot
  src="/docs/using-the-app/portfolio/dashboard-overview.png"
  alt="Portfolio dashboard overview"
  caption="The Portfolio dashboard with total value, positions, and performance chart"
/>

The Portfolio page has four main sections:

| Section | What it shows |
|---------|--------------|
| **Total Value** | Combined value of all positions + wallet balance |
| **Active Positions** | Each pool/protocol where you have funds deployed |
| **Performance Chart** | Historical value over time (24h, 7d, 30d, All) |
| **Activity Log** | All transactions: deposits, withdrawals, rebalances, harvests |

## Step-by-Step: Navigate Your Portfolio

<Steps>
  <Step>
    **Click "Portfolio" in the sidebar.** The dashboard loads with your current positions.

    <StepScreenshot
      src="/docs/using-the-app/portfolio/step-1-navigate.png"
      alt="Navigate to Portfolio"
      step={1}
      caption="Click Portfolio in the left sidebar"
    />
  </Step>
  <Step>
    **View total value.** The top card shows your total portfolio value in USD, including:
    - Wallet balance (uninvested)
    - Deployed positions (across all protocols)
    - Pending rewards (unclaimed)

    <StepScreenshot
      src="/docs/using-the-app/portfolio/step-2-total-value.png"
      alt="Total value card"
      step={2}
      caption="Your total portfolio value with breakdown"
    />
  </Step>
  <Step>
    **Check active positions.** Each position card shows:
    - Protocol name and pool
    - Deposited amount
    - Current value (with yield)
    - APY (live)
    - Time in position

    <StepScreenshot
      src="/docs/using-the-app/portfolio/step-3-positions.png"
      alt="Active positions list"
      step={3}
      caption="Each position shows protocol, value, and live APY"
    />
  </Step>
  <Step>
    **View performance chart.** Toggle between time ranges (24h, 7d, 30d, All) to see how your portfolio value has changed.

    <StepScreenshot
      src="/docs/using-the-app/portfolio/step-4-performance-chart.png"
      alt="Performance chart"
      step={4}
      caption="Track your portfolio growth over time"
    />
  </Step>
  <Step>
    **Check activity log.** Scroll down to see all transactions: deposits, withdrawals, rebalances, reward claims, and bridge transfers.

    <StepScreenshot
      src="/docs/using-the-app/portfolio/step-5-activity-log.png"
      alt="Activity log"
      step={5}
      caption="Full transaction history with timestamps and amounts"
    />
  </Step>
</Steps>
