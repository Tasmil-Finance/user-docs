<TLDR>
The Performance section shows your yield over time, high-water mark (HWM) tracking, and complete transaction history. See exactly how much you've earned, when rebalances happened, and how each protocol contributed to your returns.
</TLDR>

## Key Metrics

| Metric | What it means |
|--------|--------------|
| **Total Yield** | Cumulative earnings since first deposit |
| **APY (Realized)** | Actual annualized return based on your history |
| **High-Water Mark** | Highest portfolio value ever reached |
| **Drawdown** | Current distance below HWM (if any) |
| **Time Weighted Return** | Performance adjusted for deposit/withdrawal timing |

## Transaction History

<Steps>
  <Step>
    **Scroll to Activity section.** All transactions are listed chronologically.

    <StepScreenshot
      src="/docs/using-the-app/portfolio/performance/step-1-activity.png"
      alt="Activity section"
      step={1}
      caption="Full transaction history"
    />
  </Step>
  <Step>
    **Filter by type.** Use filters to show only specific transaction types:
    - Deposits
    - Withdrawals
    - Rebalances
    - Harvests (reward claims)
    - Bridge transfers

    <StepScreenshot
      src="/docs/using-the-app/portfolio/performance/step-2-filter.png"
      alt="Transaction filters"
      step={2}
      caption="Filter transactions by type"
    />
  </Step>
  <Step>
    **Click a transaction for details.** See the full breakdown: amounts, fees, pool, and Stellar transaction hash (clickable link to explorer).

    <StepScreenshot
      src="/docs/using-the-app/portfolio/performance/step-3-tx-detail.png"
      alt="Transaction detail"
      step={3}
      caption="Click any transaction to see full details and explorer link"
    />
  </Step>
</Steps>
