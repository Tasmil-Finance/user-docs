<TLDR>
Tasmil has four main features: **Chat** (talk to AI agents for any DeFi operation), **Aggregate** (swap and bridge), **Farming** (earn yield with AI management), and **Portfolio** (track your money).
</TLDR>

## How They Work Together

| Feature | Action | Destination |
|---------|--------|-------------|
| **Chat** (AI Agents) | execute → | Blend / Soroswap / Aquarius / Phoenix / SDEX |
| **Aggregate** (Swap & Bridge) | trade → | Blend / Soroswap / Aquarius / Phoenix / SDEX |
| **Farming** (Yield Vaults) | deploy → | Blend / Soroswap / Aquarius / Phoenix / SDEX |
| All Protocols | track → | **Portfolio** (Track Everything) |

- **Chat** is the AI-powered interface — ask in plain English and agents execute on any protocol
- **Aggregate** is the direct swap/bridge UI — no AI needed, just pick tokens and amounts
- **Farming** is the automated yield engine — set a mode and let the AI manage your positions
- **Portfolio** aggregates everything — see all positions, yields, and history in one place
