## 2026-05-18 — Vault History & Protocol Rewards

- **New:** Protocol reward tracking added to activity feed. BLND emissions now show claims with USD value.
- **New:** Performance chart now shows cumulative rewards line separate from principal.
- **New:** High Water Mark line on the performance chart.

## 2026-05-10 — Multi-Asset Support

- **New:** XLM vaults now supported alongside USDC. Fund both assets and track them side-by-side.
- **New:** Base asset selector in portfolio. Switch between USDC and XLM views.
- **New:** Presets now return per-asset pool sets (USDC pools and XLM pools are separate).

## 2026-05-01 — Mainnet Launch

- **New:** Tasmil Finance live on Stellar mainnet.
- **New:** Smart Wallet mode with Safe, Balanced, and Aggressive presets.
- **New:** Chat interface with 13 AI agents (protocol, research, bridge families).
- **New:** Aggregate surface (swap + bridge).
- **New:** Portfolio dashboard with positions, performance chart, and activity feed.
- **Protocols:** Blend, Soroswap, Aquarius, Phoenix strategies deployed.
