<TLDR>
A public SDK for developers to integrate Tasmil vault management, agent chat, and portfolio tracking into their own applications is on the roadmap. Not yet available. Check the Roadmap for updates.
</TLDR>

The Tasmil SDK will expose programmatic access to:
- Vault creation and management
- Agent chat and tool invocation
- Portfolio data and performance history
- Strategy discovery and pool data

Interested developers can watch the [Roadmap](/docs/reference/roadmap) (Later column) or reach out with use-case requests.
